CRITICAL vs HIGH — hai mức, hai hành động

CRITICAL — Block ngay

Lỗi có thể bị khai thác trực tiếp, gây mất tiền hoặc RCE.

  • Pre-commit block — commit không tạo được
  • GitLab CI fail — MR không merge được
  • Slack alert tới #dev-sec-ops

Ví dụ CRITICAL rules

security-sql-fstringf-string vào SQL
security-hardcoded-secretAPI key hardcoded
security-balance-check-no-lockRace condition balance
security-eval-injectioneval(userInput)
security-command-injection-jsexec(userInput)
security-weak-hash-androidMessageDigest MD5

HIGH — Warn, không block

Lỗi cần fix nhưng không block ngay — dev review và quyết định.

  • Pre-commit pass — commit vẫn tạo được
  • GitLab CI warn — MR vẫn merge được
  • Finding hiển thị trong MR comment

Backend security — FastAPI · SQLAlchemy · crypto

# ❌ SQL Injection — CRITICAL
db.execute(f"SELECT * FROM users WHERE id = {user_id}")
# ✅ Fix
db.execute(text("SELECT * FROM users WHERE id = :uid"), {"uid": user_id})

# ❌ Race Condition — CRITICAL
balance = get_wallet_balance(conn, uid)  # no FOR UPDATE
if balance >= amount:
    conn.execute("UPDATE wallets SET balance...")
# ✅ Fix
with conn.begin():
    row = conn.execute(text("SELECT balance ... FOR UPDATE"))
    if row.balance >= amount:
        conn.execute(...)

# ❌ Hardcoded secret — CRITICAL
WITHDRAWAL_SIGNING_KEY = "wdr_k3y_@CEX2025!"
# ✅ Fix
WITHDRAWAL_SIGNING_KEY = os.environ["WITHDRAWAL_SIGNING_KEY"]
# ❌ Weak hash — CRITICAL
hashlib.md5(pin.encode()).hexdigest()
# ✅ Fix
argon2.hash(pin)  # server-side

# ❌ Float for money — HIGH
def process(amount: float): ...
# ✅ Fix
def process(amount: str):
    d = Decimal(amount)

# ❌ Log sensitive — HIGH
logger.info("user=%s auth=%s", uid, authorization)
# ✅ Fix

# ❌ Command Injection — CRITICAL
exec(userInput)             // shell always on
execSync(req.body.cmd)
spawn(cmd, [], {shell: true})
// ✅ Fix
execFile('git', ['log', '--oneline'])  // no shell

// ❌ Eval RCE — CRITICAL

// ✅ Fix: không có use case hợp lệ

// ❌ Weak hash — CRITICAL
crypto.createHash('md5').update(data).digest('hex')
crypto.createHash('sha1').update(data)
// ✅ Fix
crypto.createHash('sha256').update(data).digest('hex')
// ❌ SHA256 for password — CRITICAL
crypto.createHash('sha256').update(password).digest('hex')
// ✅ Fix
await argon2.hash(password)
await bcrypt.hash(password, 12)

// ❌ Float for money — HIGH
const amount = parseFloat(amountStr)
const fee = Number(req.body.fee)
// ✅ Fix
const amount = new Decimal(amountStr)

// ❌ AES-ECB — HIGH
crypto.createCipheriv('aes-128-ecb', key, null)
// ✅ Fix
crypto.createCipheriv('aes-256-gcm', key, crypto.randomBytes(12))

// ❌ Static IV — HIGH
const iv = Buffer.alloc(16)  // zero IV
// ✅ Fix
const iv = crypto.randomBytes(12)

// ❌ CC_MD5 / CC_SHA1 — CRITICAL
CC_MD5(data, CC_LONG(data.count), &digest)

// ✅ Fix

Trigger: Merge Request only — không chạy khi push feature branch.

CRITICAL→ pipeline fail
